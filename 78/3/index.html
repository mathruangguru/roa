Key Concept: Vieta's Formulas / Konsep Kunci: Rumus Vieta

For a quadratic equation $ax^2 + bx + c = 0$ with roots $\alpha$ and $\beta$:

Untuk persamaan kuadrat $ax^2 + bx + c = 0$ dengan akar-akar $\alpha$ dan $\beta$:

  • Sum of roots / Jumlah akar: $\alpha + \beta = -\frac{b}{a}$
  • Product of roots / Hasil kali akar: $\alpha \cdot \beta = \frac{c}{a}$

1. Solution / Pembahasan:

Identify coefficients: $a=2, b=-8, c=5$.
Identifikasi koefisien: $a=2, b=-8, c=5$.

Sum of Roots Formula ($\alpha + \beta$): $-\frac{b}{a} = -\frac{-8}{2} = \frac{8}{2} = 4$.
Rumus Jumlah Akar ($\alpha + \beta$): $-\frac{b}{a} = -\frac{-8}{2} = \frac{8}{2} = 4$.

Product of Roots Formula ($\alpha \beta$): $\frac{c}{a} = \frac{5}{2} = 2.5$.
Rumus Hasil Kali Akar ($\alpha \beta$): $\frac{c}{a} = \frac{5}{2} = 2.5$.

Final Answer: Sum = 4, Product = 2.5

Identifikasi koefisien dari $x^2 - 6x + 2 = 0 \Rightarrow a=1, b=-6, c=2$.

Calculate Vieta: $\alpha + \beta = -\frac{-6}{1} = 6$ and $\alpha\beta = \frac{2}{1} = 2$.
Hitung Vieta: $\alpha + \beta = -\frac{-6}{1} = 6$ dan $\alpha\beta = \frac{2}{1} = 2$.

Equate the denominators of the expression: $\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\beta + \alpha}{\alpha\beta}$.
Samakan penyebut bentuk yang dicari: $\frac{1}{\alpha} + \frac{1}{\beta} = \frac{\beta + \alpha}{\alpha\beta}$.

Substitute values: $\frac{6}{2} = 3$.
Substitusi nilai: $\frac{6}{2} = 3$.

Final Answer: 3
