Solutions & Discussion

Problem 1

Concept: Grouping and cancelling (Pengelompokan dan eliminasi).

(EN) Look at the structure: $+65 - 54 + 45 - 34$.
Group the pairs: $(65 - 54) + (45 - 34) = 11 + 11 = 22$.

(ID) Perhatikan strukturnya: $+65 - 54 + 45 - 34$.
Kelompokkan pasangannya: $(65 - 54) + (45 - 34) = 11 + 11 = 22$.

22

Problem 2

Concept: Rounding to nearest 10/100 (Pembulatan ke puluhan/ratusan terdekat).

(EN) Rewrite $9$ as $10 - 1$, $99$ as $100 - 1$, and $999$ as $1000 - 1$.
Sum = $(10 + 100 + 1000) - 3 = 1110 - 3 = 1107$.

(ID) Tulis ulang $9$ sebagai $10 - 1$, $99$ sebagai $100 - 1$, dan $999$ sebagai $1000 - 1$.
Jumlah = $(10 + 100 + 1000) - 3 = 1110 - 3 = 1107$.

1107

Problem 3

Total = $450 + 81 = 531$.

(ID) Ada 9 angka yang berakhiran 9.
Jumlah angka satuan: $9 \times 9 = 81$.
Jumlah angka puluhan: $10+20+30+40+50+60+70+80+90 = 450$.
Total = $450 + 81 = 531$.

Problem 4

Concept: Gaussian Summation (Penjumlahan Gauss/Pasangan).

(EN) Pair the first and last numbers: $1+21=22$, $2+20=22$...
There are 21 numbers. Sum = $\frac{21 \times 22}{2} = 21 \times 11 = 231$.

(ID) Pasangkan angka pertama dan terakhir: $1+21=22$, $2+20=22$...
Ada 21 angka. Jumlah = $\frac{21 \times 22}{2} = 21 \times 11 = 231$.

231

Problem 13

(EN) In the units column: $A+B+C$ must end in $C$. This implies $A+B=10$ (carry 1).
In the tens column: $1 (\text{carry}) + A + B + C = 1 + 10 + C = 11 + C$. This ends in $B$.
By trial: If $A=1, B=9, C=8$. $11 + 99 + 88 = 198$. Matches $ABC$.

(ID) Di kolom satuan: $A+B+C$ berakhiran $C$. Ini berarti $A+B=10$ (simpan 1).
Di kolom puluhan: $1 (\text{simpanan}) + A + B + C = 1 + 10 + C = 11 + C$. Ini berakhiran $B$.
Dengan mencoba: Jika $A=1, B=9, C=8$. $11 + 99 + 88 = 198$. Cocok dengan $ABC$.

Problem 16

Concept: Queue Logic (Logika Antrean).

(EN) Left part up to Leader: 10 animals.
Right part from Vice: 7 animals.
If they are adjacent (Leader then Vice), total = $10 + 7 = 17$ animals.
If they overlap (Vice then Leader), total = $10 + 7 - 2 = 15$.
Standard "next to each other" usually implies adding the groups with no gap.

(ID) Bagian kiri sampai Pemimpin: 10 hewan.
Bagian kanan dari Wakil: 7 hewan.
Jika bersebelahan (Pemimpin lalu Wakil), total = $10 + 7 = 17$ hewan.
