Questions: Divisibility & Remainders

1. If the 6-digit number $472AB8$ is divisible by 88, find the value of $A - B$.

2. Find the smallest four-digit number that leaves a remainder of 3 when divided by 5, a remainder of 5 when divided by 7, and a remainder of 7 when divided by 9.

3. What is the units digit of the sum $13^{2023} + 17^{2024} + 19^{2025}$?

4. The 5-digit number $7A52A$ is divisible by 9. What is the value of A?

5. What is the remainder when $2^{100}$ is divided by 7?

6. The five-digit number $1234x$ is divisible by 8. What is the value of $x$?

7. Find the smallest positive integer that leaves a remainder of 1 when divided by 2, a remainder of 2 when divided by 3, and a remainder of 3 when divided by 4.

8. If the 8-digit number $123456AB$ is divisible by both 8 and 9, what are all possible values for $A+B$?

9. What is the units digit of $2022^{2023} \times 2023^{2024}$?

10. A number $N$ leaves a remainder of 9 when divided by 10, a remainder of 8 when divided by 9, ..., and a remainder of 1 when divided by 2. Find the smallest such positive integer $N$.

11. The 5-digit number $7A6B2$ is divisible by 72. Find the sum of all possible values of $A+B$.

12. What is the remainder when $(1! + 2! + 3! + \dots + 50!)$ is divided by 15?

13. How many simplified proper fractions are there with a denominator of 120?

14. What is the units digit of $1! + 3! + 5! + \dots + 99!$?

15. A 4-digit number leaves a remainder of 2 when divided by 5, 7, and 11. What is the smallest such 4-digit number?

16. The 5-digit number $5A83B$ is divisible by 99. Find the value of $B-A$.

17. Find the remainder when the number formed by writing the digit 7 for 2023 times (i.e., 777...7) is divided by 9.

18. What is the units digit of $7^{7^7}$?

19. The number $8765A3B$ is divisible by 72. Find the value of $2A+B$.

20. A positive integer $n$ leaves a remainder of 7 when divided by 9. What is the remainder when $4n + 3$ is divided by 9?

Solutions: Detailed Explanations

1. Solution: (Corrected)

Concept: For a number to be divisible by 88, it must be divisible by its prime factors, 8 and 11.
Rule for 8: A number is divisible by 8 if its last three digits form a number divisible by 8. A more advanced trick is that for a 3-digit number $H T U$, it's divisible by 8 if $4H + 2T + U$ is divisible by 8.
Rule for 11: A number is divisible by 11 if the alternating sum of its digits is a multiple of 11.

Step 1 (Divisibility by 8): The number is $472AB8$. The last three digits are $AB8$. Let's use the advanced rule: $4A + 2B + 8$ must be divisible by 8. Since 8 is divisible by 8, we can simplify this to: $4A + 2B$ must be divisible by 8. Dividing by 2, we find that $2A + B$ must be a multiple of 4.

Step 2 (Divisibility by 11): We calculate the alternating sum of the digits of $472AB8$. Start from the right: $(8) - (B) + (A) - (2) + (7) - (4) = A - B + 9$. This sum must be a multiple of 11.

Step 3 (Combine Rules and Find A and B): From Step 2, $A - B + 9$ could be 0, 11, -11, etc.
- If $A - B + 9 = 0 \implies B - A = 9$. The only digits that satisfy this are $B=9, A=0$. Let's test this pair with our rule from Step 1: $2A + B = 2(0) + 9 = 9$. Is 9 a multiple of 4? No. So this case is invalid.
- If $A - B + 9 = 11 \implies A - B = 2$. This is a strong possibility. Let's call this Equation (i).
- If $A - B + 9 = 22 \implies A - B = 13$. Impossible for single digits.
- If $A - B + 9 = -11 \implies A - B = -20$. Impossible. So we must have $A - B = 2$, which means $A = B + 2$.

Step 4 (Substitute and Solve): Now we substitute $A = B + 2$ into our rule from Step 1 ($2A + B$ must be a multiple of 4).
$2(B + 2) + B = 4k$ (where k is an integer)
$2B + 4 + B = 4k$
$3B + 4 = 4k$ Since 4 is a multiple of 4, for $3B + 4$ to be a multiple of 4, $3B$ must also be a multiple of 4. As 3 and 4 share no common factors, B itself must be a multiple of 4.

Step 5 (Check Possible Values for B):
- If B=0, then A = 0 + 2 = 2. Pair: (A,B)=(2,0). $A-B = 2$.
- If B=4, then A = 4 + 2 = 6. Pair: (A,B)=(6,4). $A-B = 2$.
- If B=8, then A = 8 + 2 = 10. (Not a single digit). Both valid solutions give the same result.

Final Answer: 2

2. Solution:

Concept (Consistent Shortfall & LCM): When a number leaves remainders that are consistently less than the divisors by the same amount, we can simplify the problem using the Least Common Multiple (LCM).
Let N be the number.
$N \div 5$ leaves remainder 3. This is 2 less than 5.
$N \div 7$ leaves remainder 5. This is 2 less than 7.
$N \div 9$ leaves remainder 7. This is 2 less than 9.

Step 1: Observe that in each case, the number N is 2 less than a multiple of the divisor. This means that if we add 2 to N, the new number (N+2) will be perfectly divisible by 5, 7, and 9.

Step 2: We need to find a number that is a multiple of 5, 7, and 9. The smallest such positive number is the Least Common Multiple, LCM(5, 7, 9). Since 5, 7, and 9 share no common factors, their LCM is their product: $5 \times 7 \times 9 = 315$.

Step 3: So, $N+2$ must be a multiple of 315. This means $N+2$ could be 315, 630, 945, 1260, 1575, and so on.

Step 4: We are looking for the smallest four-digit number N. We can find the corresponding N for each multiple:
- If $N+2 = 945$, then $N = 945 - 2 = 943$ (This is a 3-digit number).
- If $N+2 = 1260$, then $N = 1260 - 2 = 1258$ (This is a 4-digit number). Since this is the first 4-digit number we found, it must be the smallest.

Final Answer: 1258

6. Solution:

Concept (Divisibility by 8): A number is divisible by 8 if the number formed by its last three digits is divisible by 8.

Step 1: The number is $1234x$. The last three digits form the number $34x$. We need to find a digit $x$ (from 0 to 9) such that $34x$ is divisible by 8.

Step 2: Let's test values. We can start by dividing 340 by 8.
$340 ÷ 8 = 42$ with a remainder of 4.
This means $340$ is 4 less than a multiple of 8. To make it a multiple of 8, we need to add 4. So, the number must be $340 + 4 = 344$.

Step 3: If the number is 344, then x = 4. Let's check if there are any other possibilities. The next multiple of 8 would be $344 + 8 = 352$. In this case, x would be 2, but the tens digit would be 5, not 4. So, $352$ is not of the form $34x$.

Step 4: Therefore, the only possible value for x is 4. The question asks for the value of $x$, not the sum of values.

Final Answer: 4
